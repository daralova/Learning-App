## 📌 О проекте

**Автоматизированная обучающая система по информатике** — это приложение для изучения и практики перевода чисел между различными системами счисления.

Проект разработан в рамках курсовой работы по дисциплине «Объектно-ориентированное программирование» (АПИ НГТУ, группа АСПМ 22-1).

### 🎯 Цель приложения
- Изучить теоретические основы систем счисления
- Научиться переводить числа между двоичной, восьмеричной, десятичной и шестнадцатеричной системами
- Проверить полученные знания через тестирование

## ⚙️ Функциональные возможности

### 1️⃣ Теоретический раздел (`Form2`)
- Подробное описание систем счисления:
  - Двоичная (основание 2)
  - Десятичная (основание 10)
  - Восьмеричная (основание 8)
  - Шестнадцатеричная (основание 16)
- Правила перевода между системами
- Примеры преобразований

### 2️⃣ Преобразователь (`Form3`)
Поддерживаются **6 направлений конвертации**:
- Десятичная → Двоичная
- Десятичная → Восьмеричная
- Десятичная → Шестнадцатеричная
- Двоичная → Десятичная
- Восьмеричная → Десятичная
- Шестнадцатеричная → Десятичная

**Особенность:** каждый шаг преобразования показывается пользователю:
Преобразование 56 в двоичную систему:
Делим 56 на 2: остаток 0
Делим 28 на 2: остаток 0
Делим 14 на 2: остаток 0
Делим 7 на 2: остаток 1
Делим 3 на 2: остаток 1
Делим 1 на 2: остаток 1
Результат: 111000


### 3️⃣ Контрольный тест (`Form4`)
- 15 вопросов по теории и практике систем счисления
- Три варианта ответа на каждый вопрос
- Автоматический подсчёт баллов
- Обратная связь после каждого ответа

**Пример вопроса:**
> *«Переведите число 8A из шестнадцатеричной в десятичную систему счисления»*
> - 128 ❌
> - 138 ✅
> - 140 ❌

## 🚀 Установка и запуск

### Требования
- Windows 7 / 8 / 10 / 11
- [.NET Framework 4.5](https://dotnet.microsoft.com/en-us/download/dotnet-framework) или выше
### Способ 1: Запуск из исходников
```bash
git clone https://github.com/ВАШ_АККАУНТ/Название_репозитория.git
cd Название_репозитория/WindowsFormsApp1
# Откройте WindowsFormsApp1.sln в Visual Studio
# Нажмите F5 для запуска

### Способ 2: Запуск готового приложения
# Скачайте папку bin/Release
# Запустите WindowsFormsApp1.exe
⚠️ Важно: Приложение работает только на Windows (использует Windows Forms). Для запуска на macOS/Linux используйте Wine или виртуальную машину.
